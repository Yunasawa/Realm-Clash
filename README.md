# REALM CLASH (C/C++ TCP).

## Build

- First, compile the project:

```bash
make
```

- Start the server with:

```bash
./server
```
Server will run on port 5874

- Run the client with:

```bash
./client <ip>
```

Where <ip> is the absolute IP address of the server.

Example:

```bash
./client 192.168.1.10
```
